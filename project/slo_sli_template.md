# SLI- SLO Requirement

| **Category** | **SLI**                                                    | **SLO **                                                                                                     | **** | **** | **** | **** | **** | **** | **** |
|--------------|------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|------|------|------|------|------|------|------|
| Availability | Uptime %                                                   | 99%                                                                                                          |      |      |      |      |      |      |      |
